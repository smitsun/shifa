Step-by-Step Device Setup Guide

How to compile and deploy Shifa Mobile App on a physical iOS or Android phone
0

Pre-requisites Checklist

Before starting, ensure your host computer has the necessary software installed:

  • Flutter SDK: Download and extract Flutter from the official website. Ensure the path to the extracted flutter/bin directory is added to your system's Environment Variables (PATH).
  • Android Studio (for Android phone): Installed to configure SDK components and USB drivers.
  • Xcode (for iPhone - macOS only): Installed from the App Store.
  • USB Connection Cable: To connect the device to your PC.
1

Enable Developer Mode on Your Phone

A. For Android Devices:

  1. Open Settings $\rightarrow$ About Phone (or System Info).
  2. Locate the Build Number (sometimes under software details).
  3. Tap Build Number 7 times. You will see a popup saying "You are now a developer!".
  4. Go back to main Settings, find Developer Options (often under System/Additional Settings), and toggle USB Debugging to ON.

B. For iOS Devices (iPhones):

  1. Open Settings $\rightarrow$ Privacy & Security.
  2. Scroll down to the bottom to find Developer Mode.
  3. Toggle it ON and restart your phone as prompted.
  4. After restart, unlock the phone and click "Turn On" when prompted. Enter your passcode.
2

Connect the Phone to Your Computer

1. Connect your phone using the USB cable.

2. Unlock your phone. A popup will ask to authorize your computer. Check "Always allow from this computer" and click Allow.

3. Open a terminal (CMD or PowerShell) on your computer and verify that Flutter detects the device:

flutter devices

You should see your phone model in the listed devices (e.g., "SM-G990B • Android 13...").

3

Initialize Project Dependencies

Navigate to the folder where the Shifa mobile code is stored and prepare the project boilerplate:

cd c:\Projects\shifa\shifa_mobile
flutter create .
flutter pub get
⚠️ Android Gradle Build Warning
If this is the first time running a Flutter project on your PC, the command will download Gradle packages. This download happens automatically in the background but may take 2-5 minutes depending on your internet connection.
4

Deploy the Application to Your Phone

Execute the run command specifying your connected device. If you only have one device connected, simply run:

flutter run

If you have multiple devices connected, specify the device ID:

flutter run -d [DEVICE_ID]

Flutter will build the binary, transfer it, and launch the Shifa app directly on your screen!

5

Common Troubleshooting Tips

  • Device shows "Unauthorized": Unplug the cable, plug it back in, unlock the phone, and make sure to accept the USB debugging authorization dialog prompt immediately.
  • Flutter command not found: Double-check that you have added the path to your extracted Flutter SDK bin directory to the Environment Variables settings. Run flutter doctor to run checks.
  • iOS build code-signing errors: iOS devices require a developer certificate. Open the generated ios/Runner.xcworkspace file in Xcode and select your personal iCloud account in the "Signing & Capabilities" tab.
